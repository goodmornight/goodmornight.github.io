Hexo Theme:Cutie

​ 原作者 by Qu Tang

​ 翻译 by Spyxxx

cutie是一个反应敏捷的hexo主题，受简洁友好用户设计的www.linpx.com影响而制作。

简介

特征

  • 反应敏捷的设计
    • 单栏和双栏布局
    • 底部导航菜单
  • 可配置结构
    • 导航菜单（名字，链接，图标）
    • 由Disqus实现的用户评论部分
    • 谷歌分析
    • 标语
    • 签名
  • 额外的内容
    • Mathjax(一款相当强悍的在网页显示数学公式的插件)
    • Instant click(一个很小的JavaScript库，可以大大提高网站响应速度)
    • 搜索网页模版
    • Lightbox支持（一种jQuery插件）

Demo(演示)

访问我的个人网页演示。

毫无疑问第一次使用想要有个Demo网页的用户都已经准备好去做了

1.从github库里拷贝https://github.com/qutang/theme-cutie-demo（fork是github里的一种操作）

2.拷贝到你的本地机器上

3.运行hexo server

安装和使用

1.拷贝到你的hexo网站的themes(主题)文件夹里，并重命名为’cutie‘

1.在网站的_config.yml文件里关于主题的示例片段

1.一套默认图标，写好使用路径，就像/images/icon_name.svg:

  • archive
  • fun
  • home
  • notes
  • projects
  • resume
  • search
  • uncategorized

2.添加search搜索页面

​ 1.创建一个新的命名为’search‘页面

​ 2.在search页面前面，使用search布局

3.使用hexo的js插件hexo-prism-plugin使代码高亮

4.自定义post图标

除了默认分类图标以外，当你想要将自定义的图标显示在主页，把icon: path/to/your/icon写在页面前面。
